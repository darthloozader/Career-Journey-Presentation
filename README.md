# 📊 From Wrench's to Firewall's

Welcome to the repository for this interactive presentation!

## 🚀 How to View Live

The easiest way to view this presentation is directly in your browser—no downloads or installations required.

Simply **click the website link in the "About" section** on the right side of this GitHub repository page to launch the live presentation.

## ⌨️ Presentation Controls

If you are driving the show, use the following keyboard controls to navigate the scenes:

| Key | What it does |
| :--- | :--- |
| **Space** | Begin / next scene |
| **&rarr;** | Next scene |
| **&larr;** | Previous scene |
| **R** | Replay current scene |
| **1–9** | Jump to scene 1–9 |
| **0** | Jump to scene 10 |
| **-** | Jump to scene 11 |
| **T** | Return to title screen |
| **F** | Toggle fullscreen |
| **?** | Show keyboard help overlay |

> **💡 Presenter Tip:** I recommend pressing **F** to enter fullscreen mode first, then driving scene-by-scene using the **Space** bar. Take your time with the captions so you have time to read each one!

## 💻 How to Run Locally (Offline)

If you need to present without an internet connection, you can easily run this locally:

1. Click the green **`<> Code`** button at the top of this repository and select **Download ZIP**.
2. Locate the ZIP file on your computer and extract/unzip it.
3. Open the extracted folder and double-click the `index.html` file. It will automatically open and run in your default web browser.
